# Commentimation

A tool for making animations out of comments in Google Docs.

To use, open `commentimation.html` and follow the instructions.

Example of output:

[Bad Apple!! but it's Google Docs comments](http://www.youtube.com/watch?v=zNaSzQJzuG0)
